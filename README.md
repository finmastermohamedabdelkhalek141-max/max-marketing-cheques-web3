# إدارة شيكات العملاء — قسم التسويق

نظام ويب (صفحة واحدة) لإدارة شيكات وخطط سداد العملاء لقسم التسويق، يعمل بالكامل من المتصفح بدون خادم (Client-side فقط)، والبيانات تُحفظ محليًا في `localStorage`.

## هيكل المشروع

```
.
├── index.html      # الصفحة الرئيسية (البنية / HTML)
├── css/
│   └── style.css   # التنسيقات (Styles)
├── js/
│   └── script.js   # منطق التطبيق (Logic)
├── .gitignore
└── README.md
```

## طريقة التشغيل

### 1) تشغيل مباشر
افتح ملف `index.html` في أي متصفح حديث (Chrome / Edge / Firefox).

### 2) عبر GitHub Pages (يفضّل، لتفادي مشاكل تحميل السكربتات محليًا)
1. ارفع الملفات إلى مستودع على GitHub بنفس الهيكل أعلاه.
2. من إعدادات المستودع: **Settings → Pages → Branch: main → Save**.
3. بعد بضع دقائق سيظهر رابط الموقع المنشور تلقائيًا.

### 3) عبر خادم محلي بسيط (اختياري)
```bash
npx serve .
# أو
python3 -m http.server 8000
```

## الاعتماديات (Dependencies)

- مكتبة [SheetJS (xlsx)](https://github.com/SheetJS/sheetjs) لتصدير التقارير إلى Excel — يتم تحميلها عبر CDN تلقائيًا داخل `index.html`، ولا تحتاج تثبيت.
  - في حال تعذّر الاتصال بالإنترنت، يعمل التصدير تلقائيًا بصيغة CSV بديلة.

## ملاحظات

- جميع البيانات (العملاء، الشيكات، التحصيلات...) تُخزَّن محليًا في متصفح المستخدم فقط (`localStorage`)، ولا يوجد اتصال بأي قاعدة بيانات خارجية.
- عند الرفع على جهاز/متصفح آخر، تبدأ البيانات من جديد ما لم يتم تصديرها واستيرادها يدويًا.
